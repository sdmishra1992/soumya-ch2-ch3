# SOUMYA Challenge #3 
-------------------------

## Overview
Python script to get value from a nested JSON object, a function that you pass the object and a key and get back the value.

Example Inputs
--------------
object = {“a”:{“b”:{“c”:”d”}}}
key = a/b/c
value = d

object = {“x”:{“y”:{“z”:”a”}}}
key = x/y/z
value = a
